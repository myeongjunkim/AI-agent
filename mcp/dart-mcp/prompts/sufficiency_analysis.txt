수집된 DART 공시 정보가 사용자 질의에 답변하기에 충분한지 평가해주세요.

## 평가 기준
1. **정보 완전성**
   - 질의의 모든 측면이 커버되었는가?
   - 필요한 기업/기간/문서유형이 포함되었는가?

2. **정보 품질**
   - 관련성이 높은 문서인가?
   - 최신 정보가 포함되었는가?
   - 신뢰할 수 있는 출처인가?

3. **정보 깊이**
   - 단순 목록이 아닌 실질적 내용이 있는가?
   - 구체적인 수치나 날짜가 포함되었는가?

## 입력 정보
- 사용자 질의: {query}
- 수집 문서 수: {doc_count}
- 기업 커버리지: {companies_covered}/{companies_expected}
- 날짜 커버리지: {date_coverage}%
- 문서유형 매칭: {doc_type_match}%

## 출력 형식
```json
{
  "is_sufficient": true/false,
  "confidence_score": 0.0-1.0,
  "missing_aspects": [
    "특정 기간 데이터 부족",
    "주요 기업 정보 누락"
  ],
  "recommendations": [
    "추가 검색 제안",
    "검색 범위 확대 제안"
  ],
  "summary": "충분성 평가 요약"
}
```
